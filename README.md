# TikTok Archiver Suite

![Python](https://img.shields.io/badge/python-3.8+-blue.svg)
![Playwright](https://img.shields.io/badge/playwright-sync-green.svg)
![License](https://img.shields.io/badge/license-MIT-purple.svg)

Мощный инструмент на стыке автоматизации браузера и API-парсинга, предназначенный для полной архивации профилей TikTok. Скрипт скачивает видео (без водяных знаков), слайдшоу, аудиодорожки, аватарки, подробные метаданные пользователей/публикаций и даже древовидную структуру комментариев.

## ВНИМАНИЕ: НЕ полноценный продукт. Скрипт НЕ умеет обходить капчи, и багнутая система cookies. НЕ воспринимайте в серьёз.

---

## ✨ Основные фичи

* **Анти-Блок & Стелс-режим**: Автоматический обход систем детекции автоматизации (`navigator.webdriver`), подмена отпечатков WebGL, эмуляция реального железа и человеческого скроллинга страницы.
* **Гибридный парсинг**: Основной забор медиа идет через **TikWM API** (без водяных знаков). Если API "устало", скрипт бесшовно переключается на **резервный парсинг DOM** через JSON-гидратор (`__UNIVERSAL_DATA_FOR_REHYDRATION__`).
* **Импорт сессий (Куки)**: Поддержка загрузки `cookies.json`. Скрипт валидирует наличие `sessionid` и предупреждает, если вы работаете в гостевом режиме.
* **Скачивание Фото-постов**: Корректно обрабатывает слайдшоу (скачивает все картинки из галереи, фоновую музыку и `live_photo` при наличии).
* **Древовидные комментарии**: Рекурсивно собирает комментарии и ответы на них, сохраняя иерархию, лайки и данные авторов.
* **Сетевой мониторинг**: Интегрированный перехватчик событий сети (XHR/Fetch) и проброс консоли браузера прямо в ваш терминал для удобного дебага.

## 📂 Структура сохраняемых данных

Для каждого целевого аккаунта создается отдельная папка, структура которой выглядит следующим образом:

```text
📂 [username]/
├── 📄 user.json          # Полная статистика профиля (подписчики, лайки, соцсети)
├── 📄 likes.json         # Статус приватности лайков пользователя
├── 📄 repost.json        # Заготовка под репосты
├── 🖼️ userpic.jpg        # Аватарка в максимальном качестве
├── 🎬 [video_id].mp4     # Видео без водяного знака
├── 🖼️ [video_id].jpg     # Статичное превью (обложка)
├── 🖼️ [video_id].webp    # Динамическое превью
├── 📄 [video_id].json    # Метаданные публикации, статистика и дерево комментариев
└── 📂 [slideshow_id]/    # Если публикация — это слайдшоу
    ├── 📄 [slideshow_id].json
    ├── 🎵 audio.mp3      # Музыкальный трек из слайдшоу
    ├── 🖼️ photo1.jpg
    └── 🖼️ photo2.jpg
